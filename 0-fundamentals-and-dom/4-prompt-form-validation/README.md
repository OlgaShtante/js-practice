# Prompt Form Validation

Gathers name fields, age, and two yes/no answers via `prompt`/`confirm`,
looping until each input is valid, then derives age-in-days, age-in-5-years and
retirement status and reports them in a single `alert`.

**Practises:** dialog boxes, `while` validation loops, regex `match`,
`parseInt` / `isNaN`.

**Source:** [`main.js`](main.js)

**Run:** open [`index.html`](index.html) and click the button.
