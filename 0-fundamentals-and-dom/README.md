# Fundamentals & DOM

The earliest tasks: reading inputs, handling events, manipulating the DOM, and
styling the page (inline styles, class switching, CSS custom properties), plus
basic form input and validation.

- [`0-simplest-calculator`](0-simplest-calculator) — add two inputs on click.
- [`1-background-color-switcher`](1-background-color-switcher) — change the page
  background five different ways.
- [`2-color-list-manager`](2-color-list-manager) — add/remove colours in a `<select>`.
- [`3-color-items-editor`](3-color-items-editor) — render/add/select/delete colour
  cards from a data array.
- [`4-prompt-form-validation`](4-prompt-form-validation) — a `prompt`/`confirm`
  survey with validation loops.
