# Simplest Calculator

Reads two numbers from input fields and shows their sum in an `alert` when the
button is clicked.

**Practises:** DOM lookup, click handling, `input.value`, `parseInt`,
template literals.

**Source:** [`src/modules/main.js`](src/modules/main.js)

**Run:** serve the folder statically (see the
[repo README](../../README.md#running-a-task)).
