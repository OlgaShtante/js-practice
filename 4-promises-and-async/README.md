# Promises & Async/Await

Asynchronous JavaScript: hand-written promises, `async`/`await`, and wrapping
callback-based `XMLHttpRequest` / `fetch`.

- [`0-custom-promise-delay`](0-custom-promise-delay): a `delay(ms)` built on a
  Promise, chained with `.then`.
- [`1-promise-returning-function`](1-promise-returning-function): resolving a
  promise with a function/value.
- [`2-async-await-fetch`](2-async-await-fetch): the capstone: the same timed task
  through XHR, promises, `async`/`await`, `setTimeout`, `fetch` and a click.
- [`3-promisified-xhr`](3-promisified-xhr): wrapping a callback XHR in a Promise.
- [`4-xhr-json-callback`](4-xhr-json-callback): XHR with a callback and `JSON.parse`.
