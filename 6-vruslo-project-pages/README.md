# vRUSLO Project Pages

**Not standalone exercises.** These are link/landing pages for a separate course
project called **vRUSLO**. Each `index.html` points to external Google Sheets or
Docs (requirements, status, reports, peer review) for a stage of that project.
Several reference a `main.js` that doesn't exist here (dead script tag); the
real code lived in the separate vRUSLO project.

- [`0-vruslo-promises-links`](0-vruslo-promises-links): Promises stage.
- [`1-vruslo-events-links`](1-vruslo-events-links): Events stage.
- [`2-vruslo-url-params-links`](2-vruslo-url-params-links): URL-parsing stage.
- [`3-vruslo-xhr-links`](3-vruslo-xhr-links): XHR stage.
- [`4-vruslo-json-server-links`](4-vruslo-json-server-links): forms / JSON server.
- [`5-vruslo-peer-review`](5-vruslo-peer-review): peer-review results.
