# Image Carousel

An auto-advancing slideshow (every 3 s) that also reveals **prev/next arrows**
when you hover the carousel: hovering pauses the auto-rotation (`clearTimeout`)
and lets you step through slides manually; leaving resumes it.

**Practises:** combining automatic (`setTimeout`) and manual navigation,
`clearTimeout`, `mouseover`/`mouseleave`, index wrap-around in both directions.

**Source:** [`src/modules/main.js`](src/modules/main.js)

> Builds on [`auto-slideshow`](../2-auto-slideshow).

**Run:** serve statically (see the
[repo README](../../README.md#running-a-task)).
