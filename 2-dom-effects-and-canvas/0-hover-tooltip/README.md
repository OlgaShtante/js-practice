# Hover Tooltip

Shows a hidden message `<div>` while the pointer is over a link and hides it
again on `mouseleave`.

**Practises:** `mouseover` / `mouseleave` events, toggling
`style.display`.

**Source:** [`src/modules/main.js`](src/modules/main.js)

**Run:** open in StackBlitz or serve statically (see the
[repo README](../../README.md#running-a-task)).
