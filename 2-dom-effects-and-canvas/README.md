# DOM Effects & Canvas

Interactive DOM behaviours and a first taste of the Canvas API.

- [`0-hover-tooltip`](0-hover-tooltip) — show/hide a message on hover.
- [`1-link-redirect`](1-link-redirect) — intercept a click and redirect.
- [`2-auto-slideshow`](2-auto-slideshow) — auto-advancing image slideshow.
- [`3-image-carousel`](3-image-carousel) — slideshow with hover arrows for manual
  navigation.
- [`4-canvas-mouse-drawing`](4-canvas-mouse-drawing) — animated particles on a
  `<canvas>` via `requestAnimationFrame`.
